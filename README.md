# Student Productivity Tracker

A Python-based productivity tracker for students to manage study sessions, tasks, attendance, and progress reports.

## Features

* Add and track study sessions
* Create and manage tasks
* Calculate attendance percentage
* Generate productivity reports
* Store data locally using files

## Technologies Used

* Python
* File Handling
* Functions
* Modules

## Project Structure

student-productivity-tracker/

├── main.py

├── study.py

├── tasks.py

├── attendance.py

├── report.py

├── data/

└── README.md

## How to Run

1. Clone the repository

2. Open the project folder

3. Run:

python main.py
